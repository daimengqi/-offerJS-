### 题目描述
输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

### 思路:遍历分别存入两个数组再拼接

### 代码

```js
function reOrderArray(array)
{
    // write code here
    if(array.length <= 1) return array;
    let left = [], right = [];
    for(let i = 0; i < array.length; i++){
        if( array[i] % 2 == 1){
            left.push(array[i]);
        }else{
            right.push(array[i]);
        }
    }
    return left.concat(right);
}
```
